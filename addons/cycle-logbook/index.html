Cycle logbook

Papertrail-like record keeping, showing per cycle squad allocation and mission objectives over time.

Description

Cycle logbook is a single papertrail document where the team can keep a single history of events as cycles go by.

The document stores the following information, so that the team can go back in time and observe how they are doing:

  • Active missions (started or in-progress) on each cycle
  • Mission primary and secondary objectives
  • Hero squads allocation
  • Guardians allocation
  • Mission progress report

Cycles are logged from more recent to older, to keep new information to the top. From time to time, the team can review the log looking for opportunities to improve structure and processes.
